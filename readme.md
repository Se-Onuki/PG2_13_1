# PG2 LC1A_05_オヌキセイヤ_評価課題
## a.操作方法

    WASD : 移動
    SPACE : 弾発射
## b.加点用件で追加したもの
- シーンのクラス化
        
        Sceneクラスを継承して、配列として保存できるようにした。
        mapを使って、任意のシーンを呼び出せるようにした。
        タイマーを用意し、遷移フレームを指定できるようにした。
        入退室に呼び出される関数を使って、処理をわかりやすく出来る。
- 弾のクラス化
        
        std::vectorを使って、無限に弾を発射できるようにした。
        レンダリングパイプラインの値を元に、画面内なのかの判断をできるようにした。
## c.追加オリジナル仕様
- Json読み込みクラスを作成。
        
        JSONファイルに対して、データを読み書きできるようにした。
- UIクラスを作成

        親子関係を持った、UIクラスを作成した。
        捜索機能や、Jsonからの読み書きなどを実装。
        継承により、ポインタとして別のクラスを持つことができる。
- 文字列描画クラスを作成。

        ワイド文字列をUnicodeとして処理し、描画できる。
        swprintf_sを使って、数値などを変更したりできる。
- Audioクラスを作成。

        mapとして、音を保存し、呼び出すことができる。
